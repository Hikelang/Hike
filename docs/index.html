Why would you choose
Hike?

Speed (runtime)

The compiler of hike programming language uses LLVM backend, that is really fast.

Easy

Hike programming language has really small amount of different conceptions in compare with other programming languages. Hike has `no` standart library, instead it uses builtin namespaces with all functions, classes and globals that are needed for developing in pure Hike.

Fast developing

Syntax and design of Hike programming language is made for fast code reading and writing, but but leaves the code fast and scalable. Therefore, a hike is a compromise between the speed of the program and the speed of its development..
